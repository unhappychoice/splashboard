## What is this

Every new shell, every `cd` into a repo — instead of a blinking cursor, you
see a dashboard of the things you actually care about: git status, CI
health, PRs awaiting review, a contribution heatmap, the moon phase. One
line in your shell rc and you're done.

The killer feature: a repo that ships `./.splashboard/dashboard.toml`
reshapes the splash to itself when you `cd` in. Competitor products
(neofetch, fastfetch, starship) don't do this.
